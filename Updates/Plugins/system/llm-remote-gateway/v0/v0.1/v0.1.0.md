## Risks & Rollback
- 風險：remote API 行為依賴 OpenAI 相容介面，若第三方服務回傳格式不完全相容可能影響 stream 解析
- 錯誤的 options / online 參數配置可能導致遠端請求失敗或無法建立 stream
- 回滾方式：停用或移除 llm-remote-gateway system plugin 並恢復舊版 llamaServer 插件即可恢復原本 remote LLM 調用流程

## Notes
- 僅提供 remote LLM API gateway 能力，不包含 local 模型或內建 server 功能
- 所有 API 呼叫採 OpenAI 相容格式以支援不同 LLM 服務提供者
- 插件內部採策略模組化設計（payload、SSE、validator、errors、types）以提升可維護性與可測試性
- 所有參數皆透過 options / online 輸入提供以避免硬編程配置
