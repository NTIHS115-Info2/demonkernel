- 新增 systemPlugins/talk-engine 插件，提供 talk.nostream 與 talk.stream 能力

- typing session 採 reference count，避免多流程誤關 typing

- 核心對話流程可透過 capability registry 直接調用 talk-engine
- Discord 端 now supports reusable typing capabilities for orchestration plugins

## Tests
